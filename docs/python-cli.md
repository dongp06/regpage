# Chạy tool tổng (Python CLI)

## Chạy

```bash
cd d:/regpage
python -m python.index
```

## Menu chính

| Số | Chức năng |
|----|-----------|
| 1 | Reg & Transfer Page (Cookie) |
| 2 | Quản lý quyền Admin (Token) |
| 3 | Reg & Transfer Page (Token) [Batch] |
| 4 | Quản lý cấu hình Cookie (`.env`) |
| 5 | Xem hướng dẫn |
| 6 | Thoát |

## Thoát / Ctrl+C

- Có thể bấm **Ctrl+C** để thoát gọn (tool đã bắt `KeyboardInterrupt`, không in traceback).
